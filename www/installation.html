Installation

How to install Nova for Windows

Nova for Windows requires MSYS2 to provide the build environment. Follow the steps below.

  1. Download the .zip file from the downloads page and extract it.
  2. You need MSYS2. Download it from msys2.org and install it.
  3. After installing, open MSYS2 MINGW64 from your Start menu.
  4. Copy the extracted folder from step 1 into C:\msys64\home\YOURUSER (replace YOURUSER with your Windows username).
  5. In the MSYS2 MINGW64 terminal, run these two commands to install the required dependencies:
    MSYS2 MINGW64
    $ pacman -S mingw-w64-x86_64-llvm
$ pacman -S mingw-w64-x86_64-ncurses
  6. Navigate to the extracted folder and run the installer:
    MSYS2 MINGW64
    $ cd extractedfolder
$ ./install.sh
    If you get "no such file or directory", run ls and cd into the folder that appears.
Installation

How to install Nova for Linux

Linux installation is straightforward — extract the archive, install one dependency, and run the installer.

  1. Download the .tar.gz file from the downloads page and extract it:
    terminal
    $ tar -xzvf nova-lang-*.tar.gz
  2. Install ncurses (required for the interactive editor):
    terminal
    $ sudo apt install libncurses-dev
  3. Navigate into the extracted folder and run the installer:
    terminal
    $ cd nova-lang-*
$ ./install.sh
  4. Restart the terminal, and start coding!
After installation, verify it worked by running n++ --version in a new terminal.
